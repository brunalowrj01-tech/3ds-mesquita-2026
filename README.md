# 3ds-mesquita-2026 

**Bibliotec**

*Nosso projeto Bibliotec é um site que será disponibilizado na escola com o objetivo de facilitar a vida dos alunos no acesso às informações da biblioteca. A plataforma permitirá pesquisar livros de forma rápida, verificar a disponibilidade dos exemplares e acompanhar empréstimos sem precisar ir até o local. Além disso, o site ajudará na organização dos estudos, tornando mais simples encontrar materiais para trabalhos e leituras indicadas pelos professores. Com o Bibliotec, os alunos terão mais praticidade, economia de tempo e acesso fácil ao conteúdo escolar em um só lugar.*
